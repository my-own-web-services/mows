## Overview

-   Written in async [Rust](https://www.rust-lang.org/) using [Tokio](https://tokio.rs/) and [Actix](https://actix.rs/)
-   Using [Trust-DNS](https://github.com/bluejekyll/trust-dns) for validation
-   Thin layer, wrapping Keydb, to validate data and to request signing records with Vault
