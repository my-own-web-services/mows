# Getting Started

The Pektin API Server exposes the most basic functions to mutate the Keydb database via an HTTP endpoint.

Following methods are supported

-   [/get](./methods#get)
-   [/get-zone-records](./methods#get-zone-records)
-   [/set](./methods#set)
-   [/search](./methods#search)
-   [/delete](./methods#delete)
-   [/health](./methods#health)
